# Online-Shop
eCommerce page with product display and shopping cart

This is a simple online store demo project demonstrating a basic shopping cart functionality using HTML, CSS, and JavaScript.
Users can add products to the cart, view the cart with total price, and clear the cart.

## Project Structure
* index.html          -----> Product listing page
* cart.html           -----> Shopping cart page
* home.html           -----> Home page or landing page
* styles.css         -----> Stylesheet for all pages
* script.js           -----> JavaScript for cart functionality

## Features

Multiple product listings with “Add to Cart” buttons

Persistent cart stored in browser localStorage

Cart icon with dynamic item count updating in real-time

Separate cart page showing all items and total price

Clear Cart button to empty the cart instantly

Responsive and user-friendly design

## How to Use

Open home.html or index.html in your browser to browse products.

Click Add to Cart on any product to add it.

The cart icon in the header updates with the current item count.

Click the cart icon or go to cart.html to see your shopping cart.

View items, total price, or clear the cart with the button.

Click "Back to shop" to continue shopping.

## Technical Details

HTML: Structure for multiple pages.

CSS: Styling with flexbox, responsive layout, and hover effects.

JavaScript:

Manages cart state via localStorage for persistence.

Updates cart count in header dynamically.

Renders cart items and calculates total price on the cart page.

Handles “Clear Cart” functionality.

## How to Run Locally

No backend or build tools needed — just open the .html files directly in your browser.

Clone or download the repository.

Open home.html or index.html in any modern browser.

Enjoy browsing and adding items to your cart!
